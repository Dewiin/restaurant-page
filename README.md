## Installation & Setup

### Prerequisites
Ensure you have Node.js and npm installed:
```sh
node -v  # Should be 18+  
npm -v
```

### Clone the Repository
```sh
git clone https://github.com/Dewiin/restaurant-page.git
cd restaurant-page
```

### Install Dependencies
```sh
npm install
```

### Run the App Locally
```sh
npm run dev
```
